# Servlet 生命周期

实例化，初始化，服务，销毁

这次我们主要关心 Servlet 的初始化方法

# Servlet 中的初始化方法

有两个：

- `init()`
- `init(config)`

分别为：

```java
// 带参数的init方法，init(config)
public void init(ServletConfig config) throws ServletException {  
    this.config = config;  
    this.init();  
}  
// 无参init方法，init()
public void init() throws ServletException {  
}
```

如果我们想要在 servlet 初始化时做一些准备工作，那么我们可以重写 `init()` 方法
