# 过滤器链

<img src=" https://coachhe-1305181419.cos.ap-guangzhou.myqcloud.com/Redis/20230219202023.png" width = "80%" />

在没有特殊指定的时候，过滤器链中的过滤器是按照类名来先后执行的，因为这里三个过滤器分别是 `Filter01`、`FilterB` 和 `FilterC`，因此首先执行的是 `Filter01`，然后是 `Filter02`，最后是 `Filter03`。

如果采用 XML 配置文件进行配置，那么按照配置的先后顺序进行执行。
