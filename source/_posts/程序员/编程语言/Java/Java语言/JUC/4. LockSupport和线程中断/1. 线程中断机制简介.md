| 方法类型       | 方法名            | 方法作用                   |
| -------------- | ----------------- | -------------------------- |
| void           | `interrupt()`     | 中断此线程                 |
| static boolean | `interrupted()`   | 测试当前线程是否已经被中断 |
| boolean        | `isInterrupted()` | 测试此线程是否已经被中断                           |

1. 一个线程不应该由其他线程来强制中断或停止，而是应该由线程自己自行停止。
