# 集群功能辅助脚本

## `xsync`

用来在多个集群之间同步数据。

```sh
#!/bin/bash 
#1 获取输入参数个数，如果没有参数则直接退出 
pcount=$# 

if((pcount==0)); then 
    echo no args; 
    exit; 
fi 

#2 获取文件名称 
p1=$1 
fname=`basename $p1` #这里就是拿到最终文件的名称，/opt/module就是module 
echo fname=$fname 

#3 获取上级目录到绝对路径 
pdir=`cd -P $(dirname $p1); pwd` #这里是得到文件的路径 
echo pdir=$pdir 

#4 获取当前用户名称 
user=`whoami`  #也就是coachhe 

#5 循环 
for((host=2;host<5;host++)); do    
    echo -------- chadoop10$host ---------
    rsync -rvl $pdir/$fname root@hadoop10$host:$pdir
done
```

## `xcall`

用来在多个集群中查询相同命令的任务

```sh
#!/bin/bash

export PATH=$PATH:$JAVA_HOME/bin

for i in hadoop102 hadoop103 hadoop104
do
	echo "=================$i==============="
	ssh $i "source /etc/profile; $*"
done

```

## `lg.sh`

用来在多个集群产生用户行为日志

```sh
#!/bin/bash

for i in hadoop102 hadoop103
do
	echo "=================$i==============="
	ssh $i "cd /root/module/applog; /root/module/tencent_jdk8/bin/java -jar gmall2020-mock-log-2021-10-10.jar >/dev/null 2>&1 &"
done
```
